# FMBT-Fatigue-Monitoring-Baseline-Toolkit-
FMBT is a high-performance research toolkit for fatigue-related studies, which bridges the gap between software paradigms and hardware sensors. FMBT 是一套高性能的疲劳驾驶监测科研基准工具包。系统集成了高强度 N-back 疲劳诱导、金标准 PVT 客观评估、以及基于 Z 分数的自动化 PIS 疲劳评分算法。内置 LSL 接口，确保软硬件数据流实现亚毫秒级的精确同步。
